# SEO_NAVER_DOMAIN_SETUP.md

## 현재 준비된 것

- `index.html`: 홈페이지 본문, 제목, 설명, 오픈그래프, 구조화 데이터 포함
- `robots.txt`: 검색 엔진 접근 허용
- `sitemap.xml`: 검색 엔진 제출용 사이트맵
- `site.webmanifest`: 사이트 이름과 색상 정보
- `assets/`: 로고와 홈페이지용 이미지 자료

## 확정 도메인

대표 도메인은 아래 주소로 확정했습니다.

- `autoleasener.co.kr`

가비아에서 등록을 완료했습니다.
GitHub Pages 공개본과 연결하기 위해 DNS 설정 및 Pages 사용자 지정 도메인 확인이 필요합니다.

## 배포 주소

대표 공개 주소:

`https://autoleasener.co.kr/`

호스팅은 기존 GitHub 저장소의 GitHub Pages를 사용합니다.
`CNAME` 파일과 검색용 메타 정보는 대표 도메인 기준으로 설정합니다.

## 네이버 검색 노출 준비 순서

1. 홈페이지를 실제 인터넷 주소에 배포합니다.
2. HTTPS 연결이 완료된 후 네이버 서치어드바이저에 접속합니다.
3. 사이트 URL `https://autoleasener.co.kr/`을 등록합니다.
4. 네이버에서 제공하는 소유권 확인 코드를 받습니다.
5. `index.html`의 주석 위치에 메타 태그를 넣거나, 네이버가 주는 인증 파일을 루트에 추가합니다.
6. `sitemap.xml`을 제출합니다.
7. `robots.txt` 수집이 정상인지 확인하고 대표 페이지 수집을 요청합니다.
8. 네이버 지도 API 설정의 웹 서비스 URL에 `https://autoleasener.co.kr`을 추가합니다.

## 검색어 방향

- 브랜드 검색어: `오토리스너`
- 채널 연계 검색어: `리스준`
- 홈페이지 제목과 설명에는 두 이름을 직접 표시합니다.
- 검색 노출 순서와 시점은 네이버 수집 및 판단에 따라 달라지므로 즉시 상위 노출을 보장할 수 없습니다.

## 파워링크와 일반 검색의 차이

- 파워링크: 결제한 키워드와 광고 심사 결과에 따라 표시되는 광고 영역입니다.
- 일반 검색: 서치어드바이저 등록, 사이트 수집, 페이지 품질을 바탕으로 노출됩니다.
- 파워링크의 최종 연결 주소는 HTTPS가 정상화된 뒤 `https://autoleasener.co.kr/`로 맞춥니다.

## 사용자가 확인해야 할 것

- 상담 전화번호 또는 카카오 링크
- 리스준 유튜브 채널 링크가 현재 채널과 맞는지
- 실제 공개할 회사 소개 문구
- 네이버 소유권 확인 방식
